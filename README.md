# Gym Static Web Project

This repository contains a static website project for a gym.

## Project Overview
The Gym Static Web Project is hosted and delivered via AWS services:

1. **Amazon S3** - Used to store the static files and serve the website content.
2. **Amazon CloudFront** - Used as a Content Delivery Network (CDN) to ensure fast and secure delivery of the website globally.

## Website URLs
- **S3 Static Website URL:** [http://gym-static-webproject-nitish.s3-website.ap-south-1.amazonaws.com/](http://gym-static-webproject-nitish.s3-website.ap-south-1.amazonaws.com/)
- **CloudFront Distribution URL:** [https://d1dnwdp8sgoruv.cloudfront.net/](https://d1dnwdp8sgoruv.cloudfront.net/)

## Features
- **Responsive Design:** Designed to work seamlessly across devices.
- **Lightweight and Fast:** Optimized for speed and performance.
- **Secure CDN Delivery:** Utilizing AWS CloudFront for secure and reliable content delivery worldwide.

## How to Access
1. Open the **S3 Static Website URL** to directly access the website hosted on AWS S3.
2. Alternatively, use the **CloudFront Distribution URL** for a faster and globally optimized experience.
